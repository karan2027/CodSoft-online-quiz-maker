- User Registration
- Secure Login
- JWT Authentication
- Forgot Password
- Email OTP Verification
- Reset Password
- Logout

## Quiz Management
- Create, Edit and Delete Quizzes
- Save Draft
- Publish Quiz
- Public & Private Visibility
- Categories & Subcategories
- Search & Filtering

## Quiz Attempt
- Timer
- Auto Submit
- Score Calculation
- Leaderboard
- Attempt History
- Result Review

## Dashboard
- My Quizzes
- My Results
- Statistics

## Security
- Password Hashing
- Protected Routes

# ⚙️ Installation

```bash
git clone <repository-url>
cd QuizMaster-Pro

cd backend
npm install

cd ../frontend
npm install
```

Run backend

```bash
cd backend
npm run dev
```

Run frontend

```bash
cd frontend
npm start
```

---

# 🔐 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=
CLIENT_URL=
MONGODB_URI=
JWT_SECRET=
JWT_EXPIRES_IN=
EMAIL_HOST=
EMAIL_PORT=
EMAIL_SECURE=
EMAIL_USER=
EMAIL_PASS=
EMAIL_FROM=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```

Never commit your `.env` file to GitHub.
